# Name-Generator
One of our family friends recently had a baby and are currently looking for a name for the little fella. This is my contribution.

## How to Use
![alt text](https://github.com/banisingh/Name-Generator/blob/master/PrevCut.gif)

Run the program on terminal. The program will then ask user for an input for 5 letters (biased decision on the lenght of a name). This can either be a vowel, consonant, or another letter (if user does not want to chose vowel, nor consontant. The Name Generator will then return 10 different names that one can chose from.
